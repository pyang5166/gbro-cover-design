# 风格六：海报拼贴风

多张参考图分层叠加，有明确前中后景关系，叙事丰富纵深感强。

## 视觉特征
- 前景：最大的素材图或人物，带景深/投影
- 中景：人物或次要素材图，与前景有遮挡
- 后景：模糊的背景素材，丰富但不抢主视觉
- 字体：压在层次之间，白色或描边效果
- 背景：深色，衬托各层次

## 固定
- 画幅：3:4 竖版
- 人物：参考图1
- 建议至少提供一张额外素材（图2）
- 关键元素距四边至少留约10%边距

## 变量
- 各元素在前/中/后景的位置
- 叠加方式
- 大字
- 背景色调
- 字体风格

## 提示词模板

```
3:4 竖版构图。

参考图1的[性别]五官特征，保持五官一致性，只要半身。

[具体表情]，[人物姿势和身体位置]，

前景：[前景元素详细描述，大小、位置、视觉细节]，占画面约[比例]，带轻微投影和浮动感，
中景：人物主体，与前景元素有遮挡关系，[人物被前景遮挡哪个部位]，
后景：[后景元素描述，轻微模糊处理，内容简述]，[数量]个，错落排列，作为背景层，
各元素之间有前后遮挡关系，制造层次和纵深，

巨大的中文大字"[封面标题]"，[字体风格]，[颜色效果，建议白色描边或纯白]，
压在人物和前景元素之间，与多个层次形成遮挡关系，

背景：[深色沉稳色调，衬托各层次]

所有关键元素集中在中间区域，距画面四边至少保留约10%边距，悬浮元素有轻微投影，层次感强
```
