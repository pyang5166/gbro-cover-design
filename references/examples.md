# 示例提示词库

8 组完整示例提示词（按真实成品封面逆向还原）。用途：生成新提示词前，找同风格（或构图最接近）的示例做 few-shot 参照，学它的**具体程度**——姿势写到手在哪、道具写到上面有什么内容、空间关系写清楚谁遮挡谁。

---

## 示例1「双面特工」— 深色渐变风

要点：人物在底部只露头和手臂，主视觉（机器人+道具）占上方大空间，四个标题字分占四角形成框架。

```
3:4 竖版构图。

参考图1的女性五官特征，保持五官一致性，只露出头部和高举的双臂。

张嘴震惊表情，眼睛瞪大，人物位于画面底部中央，双手高高举起一台银色 Mac mini 主机过头顶，
主机上方悬浮一只巨大的红色圆胖机器人，黑色圆眼睛，机械爪手臂，正俯冲向主机，带速度线和动态模糊，
机器人左侧爪子下方散落蓝色系图标：邮件、日历、VS Code、机票，代表日常好用的一面，
右侧爪子下方散落红色系图标：骷髅头、病毒虫、挂锁、一段恶意代码窗口，代表危险的一面，
左右两组图标形成蓝红对比，中间迸出橙色火花碎片，

巨大的中文大字"双""面""特""工"分别位于画面四个角落，超粗白色黑体，部分被机器人和图标遮挡，形成视觉层次感

背景：深蓝到暗红的深色渐变，柔和过渡

所有关键元素集中在中间区域，距画面四边至少保留约10%边距

悬浮元素轻微投影，高饱和度，又整齐又凌乱的美感
```

## 示例3「角色一致」— 海报拼贴风

要点：人物手势和主视觉有互动（托举），多张小卡片在后景错落排列做层次，主副标题大小分级。

```
3:4 竖版构图。

参考图1的女性五官特征，保持五官一致性，只要半身。

开心微笑表情，人物位于画面左侧，右手在胸前比 OK 手势，左手在画面右下方摊开手掌，

前景：左手掌心上方悬浮一个巨大的像素风红发角色立绘，占画面约30%，带轻微投影和浮动感，
中景：人物主体，肩部被前景像素角色轻微遮挡，
后景：6张白底像素角色卡片，同一角色的不同朝向（正面/侧面/背面），错落漂浮排列，轻微模糊和倾斜，作为背景层，
各元素之间有前后遮挡关系，制造层次和纵深，周围散落少量深紫色像素方块点缀，

巨大的中文大字"角色一致"在画面顶部，超粗白色黑体，
副标题"超简单！！"略小一行在主标题左下方，纯白色，

背景：暗红到深紫的深色渐变，衬托各层次

所有关键元素集中在中间区域，距画面四边至少保留约10%边距，悬浮元素有轻微投影，层次感强
```
