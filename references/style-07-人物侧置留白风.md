# 风格七：人物侧置留白风

人物偏向一侧，对面留大面积空间给标题，版面对比强大气。

## 视觉特征
- 人物：偏一侧，占宽25-35%，眼神指向留白区
- 留白区：占60-70%，标题文字占满，大而有力
- 背景：中性色或浅色，不抢版面
- 字体：深色为主，大号，可竖排或横排

## 固定
- 画幅：3:4 竖版
- 人物：参考图1，偏左或偏右，占25-35%
- 对侧留白充足
- 关键元素距四边至少留约10%边距

## 变量
- 人物在左还是右
- 留白背景色
- 标题排列方式（竖排/横排/大小字组合）
- 字体风格

## 提示词模板

```
3:4 竖版构图。

参考图1的[性别]五官特征，保持五官一致性，只要半身。

人物置于画面[左/右]侧，占画面宽度约30%，[具体姿势]，
[具体表情]，眼神和身体自然朝向[对侧留白区]，

画面[对侧]约65%是大面积留白，
超大中文"[封面标题]"占满留白区域，[字体风格]，[颜色：深色]，
[排列方式：竖排从上到下/横排分两行/主字大副字小等]，
[如有补充信息：一行小字灰色在标题下方]，

背景：[中性或浅色色调，整体柔和]

人物与文字之间保留足够的呼吸距离，不拥挤，所有关键元素距画面四边至少保留约10%边距
```
