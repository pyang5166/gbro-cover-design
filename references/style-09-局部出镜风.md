# 风格九：局部出镜风

只露手、半脸或侧脸，产品或文字是绝对主角，人物只是见证者。

## 视觉特征
- 产品/截图：占画面主体，清晰锐利，细节可读
- 人物：局部出镜，占边缘15-25%，不喧宾夺主
- 背景：浅色/中性，让产品清晰
- 字体：压在产品顶部或旁边，深色或白色

## 固定
- 画幅：3:4 竖版
- 人物：参考图1，只露局部
- 关键元素距四边至少留约10%边距

## 变量
- 露出的部位（手/半脸/侧脸）
- 局部的动作（点击/触碰/指向/展示）
- 产品/主体的详细呈现
- 大字
- 背景色调
- 字体风格

## 提示词模板

```
3:4 竖版构图。

画面主体是[产品/截图/卡片的详细描述：内容布局/颜色/关键信息]，
占画面约[比例]，白底或[底色]，清晰锐利，
（如有图2：主体内容即参考图2，[图2的展示方式]）

画面[右下角/左侧边缘/底部边缘]局部出镜：
参考图1的[手部特写/半张脸/侧脸轮廓]，
[具体局部动作：食指轻触/拿起/指向某个位置]，
[局部出镜的细节：手的姿势/脸的朝向/与主体的距离关系]，

"[封面标题]"，[字体风格]，[颜色效果]，压在主体[上方/左侧/下方]

背景：[浅色/中性色，让主体成为绝对焦点]

产品是焦点，人物只是见证者，所有关键元素距画面四边至少保留约10%边距
```
