# 风格八：背影构图风

人物背对镜头，不露脸，制造代入感，适合励志、开放、新开始类内容。

## 视觉特征
- 人物：背影，只见后脑、肩膀、背部，有动势
- 前方：人物朝向的目标或主视觉元素
- 背景：大气，开阔，有纵深感
- 字体：在人物上方，宽松舒展

## 固定
- 画幅：3:4 竖版
- 人物：参考图1背影，不露脸
- 关键元素距四边至少留约10%边距

## 变量
- 人物朝向和动作（举手/仰头/行走等）
- 前方/上方的主视觉元素
- 大字
- 背景色调（通常偏深，营造开阔感）
- 字体风格

## 提示词模板

```
3:4 竖版构图。

[性别]人物背对镜头，只见后脑、肩膀和背部，发丝清晰，不露脸，

[具体动作：仰头看向前方/双臂展开/缓步向前等]，[身体姿势描述，有动势感]，
人物前方[或上方]是[主视觉元素的详细描述：内容/大小/状态/动态]，
[主视觉元素和人物之间的空间关系描述]，

巨大的中文大字"[封面标题]"在人物[上方/左右两侧]，[字体风格]，[颜色效果]，
字间距宽松，与背景有足够对比，

背景：[大气深色渐变，营造开阔纵深感]

画面有[叙事感描述：释放/出发/到达等]，代入感强，所有关键元素距画面四边至少保留约10%边距
```
