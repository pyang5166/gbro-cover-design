# 风格五：极简留白风

大面积留白，文字是主视觉锤，人物轻量出镜，整体克制呼吸感强。

## 视觉特征
- 背景：浅色/白色，大面积留白
- 人物：体量小（20-25%），偏角落
- 字体：大而干净，占据留白区主体，是唯一视觉锤
- 整体：元素极少，不做点缀，克制优先

## 固定
- 画幅：3:4 竖版
- 人物：参考图1，体量小，偏一侧或下方
- 留白面积 > 60%
- 关键元素距四边至少留约10%边距

## 变量
- 人物位置（左/右/下方）
- 留白背景色（白/米白/浅灰）
- 大字位置和排列方式
- 字体风格（偏细或中等，不要超粗）

## 提示词模板

```
3:4 竖版构图。

参考图1的[性别]五官特征，保持五官一致性，半身，体量偏小。

人物置于画面[左下/右下/一侧]，占比约20%，[简单姿势]，
[轻量表情，不要过度夸张]，眼神/身体朝向留白区，

画面[对侧/上方]大面积[白色/米白/浅灰]留白，
超大中文"[封面标题]"占据留白区主体，[字体风格，偏细或中等粗细]，[深色]，
字间距宽松，呼吸感强，
[如有副标题：一行小字灰色，作为补充信息]，

背景：整体[浅色色调]，极简，无装饰

构图克制，留白是设计的一部分，所有关键元素距画面四边至少保留约10%边距
```
